# Sample CKP Package

`sample-biomechanics.ckp` is a valid `.ckp` file generated by the test suite (`SamplePackageGeneratorTests`). It contains 5 claims from a fictional biomechanics textbook across 4 domains and all 4 tiers.

## What's inside

A `.ckp` file is a standard ZIP archive. You can inspect it with any ZIP tool:

```
sample-biomechanics.ckp
├── manifest.json
├── claims/
│   ├── claims.json                     (5 claims)
│   ├── by-tier/t1.json ... t4.json     (precomputed tier slices)
│   └── by-domain/*.json                (precomputed domain slices)
├── evidence/
│   ├── citations.json                  (4 citations)
│   └── axiom-refs.json
├── structure/
│   ├── chapters.json                   (5 chapters)
│   ├── domains.json                    (4 domains)
│   └── glossary.json                   (1 entry)
└── history/
    ├── editions.json
    └── tier-changes.json
```

## Reading the package in code

```csharp
using Ckp.IO;

var reader = new CkpPackageReader();
await using var stream = File.OpenRead("sample-biomechanics.ckp");
var package = await reader.ReadAsync(stream);

Console.WriteLine($"Book: {package.Manifest.Book.Title}");
Console.WriteLine($"Claims: {package.Claims.Count}");

foreach (var claim in package.Claims)
    Console.WriteLine($"  [{claim.Tier}] {claim.Id}: {claim.Statement[..60]}...");
```
